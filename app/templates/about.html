Least Significant Bit (LSB) Steganography

Digital images are made of pixels. Each pixel has Red, Green, and Blue (RGB) values, typically ranging from 0 to 255.

The number 255 in binary is 11111111.

If we change the last bit (the "Least Significant Bit") to 0, the value becomes 254 (11111110). This change is so small that the human eye cannot see the difference in color.

AES-GCM Encryption üîê

Before we hide your message, we encrypt it using AES-GCM (Advanced Encryption Standard - Galois/Counter Mode).

This ensures two things:

  • Confidentiality: Only someone with the password can read it.
  • Integrity: If the image is tampered with, the decryption will fail instead of showing garbage data.
